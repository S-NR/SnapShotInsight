Memory Computation

Computation for Displays

Formula:
Memory (Bytes) = Width × Height × BitsPerPixel / 8
Memory (KB) = (Width × Height × (bpp / 8)) / 1024
If using double buffering, multiply the result by 2.

Examples

Resolution (WxH) Bits per Pixel (bpp) Bytes per Pixel Single Buffer (KB) Double Buffer (KB)
129 × 160 16 2 40.31 80.62
320 × 240 16 2 150 300
160 × 100 1 0.125 1.95 3.91

Step-by-step Calculation Example

Display: 320 × 240 at 16 bpp

  • Width = 320
  • Height = 240
  • Bits per pixel = 16
  • Bytes per pixel = 16 / 8 = 2

Single Buffer: 320 × 240 × 2 = 153,600 bytes ≈ 150 KB

Double Buffer: 153,600 × 2 = 307,200 bytes ≈ 300 KB

Computation for Variables or Arrays in C/C++

Variables and Arrays in C / C++

Formula:
Memory (bytes) = Number of elements × Size of each element (in bytes)

Examples

  • int arr[100]; → 100 integers
  • If int is 4 bytes → 100 × 4 = 400 bytes
  • char name[50]; → 50 bytes (1 byte per character)
  • float matrix[10][10]; → 10 × 10 × 4 = 400 bytes (if float = 4 bytes)
