1. GPIO Modes

ModeDescription
InputReads digital signal (HIGH/LOW) from external source
OutputSends digital signal to devices like LED, relay, etc.
Alternate FunctionUsed by internal peripherals (UART, SPI, I2C...)
AnalogUsed for ADC input or DAC output

2. Input Configuration Options

TypeDescription
Floating InputNo pull resistor; may pick up noise
Pull-Up InputInternally pulled HIGH when unconnected
Pull-Down InputInternally pulled LOW when unconnected

3. Output Configuration Options

TypeDescription
Push-PullDrives pin HIGH or LOW directly; most common mode
Open-DrainCan pull LOW, relies on pull-up resistor to go HIGH

4. Output Speed (e.g., STM32)

SpeedDescription
Low SpeedSlow toggling signals
Medium SpeedBalanced use; general-purpose
High / Very High SpeedFast signals (clocks, SPI), may cause EMI

5. Pin State and Voltage

StateDigital ValueVoltage (3.3V system)
LOW0~0V
HIGH1~3.3V

6. Sample GPIO Configuration (STM32 HAL)

GPIO_InitTypeDef GPIO_InitStruct = {0};

__HAL_RCC_GPIOA_CLK_ENABLE();

GPIO_InitStruct.Pin = GPIO_PIN_5;
GPIO_InitStruct.Mode = GPIO_MODE_OUTPUT_PP;
GPIO_InitStruct.Pull = GPIO_NOPULL;
GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_LOW;
HAL_GPIO_Init(GPIOA, &GPIO_InitStruct);
