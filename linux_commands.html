Linux Commands Cheat Sheet

Fast access to common and essential terminal commands

Basic Commands

CommandDescription
pwdPrint current working directory
lsList files in current directory
ls -lLong list with details
ls -aInclude hidden files
cd <dir>Change directory
cd ..Go up one level
cd ~Go to home directory
mkdir <dir>Create a directory
rm <file>Delete a file
rm -r <dir>Delete a directory recursively
cp <src> <dst>Copy file or directory
mv <src> <dst>Move or rename file or directory
touch <file>Create an empty file
cat <file>Display file contents
less <file>View file with scroll
file <file>Show file type

System Information

CommandDescription
uname -aShow system info
top or htopShow running processes
df -hDisk usage (human readable)
free -hMemory usage
uptimeHow long the system has been running
whoamiShow current user
idShow current user's UID and groups
hostnameShow or set system hostname

Package Management (Ubuntu/Debian)

CommandDescription
sudo apt updateUpdate package lists
sudo apt upgradeUpgrade installed packages
sudo apt install <pkg>Install a package
sudo apt remove <pkg>Remove a package

Permissions & Ownership

CommandDescription
chmod <perm> <file>Change file permissions
chown <user>:<group> <file>Change ownership
ls -lView permissions and ownership

Searching

CommandDescription
find <dir> -name "<pattern>"Find files
grep "<text>" <file>Search for text in a file
grep -r "<text>" <dir>Recursive search in directory

Networking

CommandDescription
ping <host>Test connectivity
ip a or ifconfigShow IP addresses
netstat -tulnList open ports
curl <url>Fetch URL content
wget <url>Download file

User Management

CommandDescription
adduser <name>Add a user
passwd <name>Set password for user
sudoRun command as root
suSwitch user

Miscellaneous

CommandDescription
echo "Hello"Print message
historyShow previous commands
clearClear terminal
man <command>Show manual for command
exitExit terminal or SSH session

Package Version via apt

apt list --installed | grep <package>List installed version
apt show <package>Show detailed package info
dpkg -s <package>Get version and status

Check if a Command/Package is Installed

which <command>e.g. which python3
command -v <command>Find if available
whereis <command>e.g. whereis gcc
type <command>Show command type
readlink -f $(which <command>)Full path

Clock Skew Fix

sudo date --set "$(date)"Sync system date
sudo hwclock --systohcSync hardware clock
find . -type f -exec touch {} +Update file timestamps
